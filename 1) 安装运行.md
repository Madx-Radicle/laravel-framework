# 安装运行

这里的环境主要是针对cpanel和php7.4, 长篇短说。

1) 到easyapache4里安装php74,然后启动以下extension
```sh
php74-php-iconv
php74-php-fileinfo
php74-php-mbstring
```

2) shell操作
```sh
# 回到账号的根目录
cd ~

# 
php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"
php composer-setup.php
php -r "unlink('composer-setup.php');"

```
